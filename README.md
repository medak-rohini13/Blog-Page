📝 Dynamic Blog Page (JavaScript Mini CMS)

A simple dynamic blog system built using HTML, CSS, and JavaScript where users can add blog posts instantly without editing the HTML file.

This project demonstrates basic DOM manipulation and state management in JavaScript, acting as a mini Content Management System (CMS).

🚀 Features
➕ Add blog posts dynamically
🧠 Title, author, description, date support
⚡ Instant UI updates (no page reload)
📌 Posts stored in memory (JavaScript state)
🎨 Clean modern UI design
📱 Responsive layout
✨ Hover effects on cards
🧰 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
